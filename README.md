# Particle Tracking

Notebook for analysing trajectories from mechanical particle simulator using image analysis.

## Requirements

Install via Anaconda:

- `scipy`
- `matplotlib`
- `pandas`
- `trackpy`

Command line tools:

- `ffmpeg`
